JavaScript language

Syntax

There are a few things you will see over and over again when you are learning to use JavaScript. Here are a few examples and their use cases:

Parentheses ()

  • They are used in functions when you create functions: function myFunction() {}
  • Or when you execute the function: myFunction()
  • You also use them to list your conditions in if statements: if (x < y){console.log( "you win!")}
  • They are needed when setting your conditions in for loops: for (var i=0; i <10; i++) {console.log(i)}
  • Or when setting the conditions in while loops: while(x
  • Or simply for order of mathematical operations x = 5 * (9+1)

Brackets []

  • You use brackets to create arrays: var shopping_list = ["milk", "butter", "apples"]
  • Then they are used to index into an array: shopping_list[1] --- this returns the second item in our array.
  • They are also used to access properties of an object: myObject["property"]

Braces {}

  • Braces are used when you create functions: function myFunction() { //make my function do stuff }
  • They are also used when creating objects: var fruit = { name: "apple", colour: "red"}

Single quotes ''

  • Single quotes are used to create strings: var str = 'I am a string'
  • They are also used when you print statements to the console: console.log('Print me')
  • Values of an object property also need to be wrapped into quotes: var dog = { name: 'Fiffy', colour: 'black'}
  • Or when you use bracket notation to access the property of an object: myObject['property']

Double quotes ""

  • Double quotes are used to create strings: var str = "I'm a string"
  • They are also used when you print statements to the console: console.log("Print me")
  • Values of an object property also need to be wrapped into quotes: var dog = { name: "Fiffy", colour: "black"}
  • Or when you use bracket notation to access the property of an object: myObject["property"]

Note that single and double quotes are interchangeable in JavaScript. When you are writing English text, it is often easier to use double quotes because a single quote may have to be escaped in sentences like var str = 'I\'m a string'. When writing for JSON, double quotes should always be used. When you are writing HTML code inside JavaScript double quotes should also be used.
